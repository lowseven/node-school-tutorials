# Install

```javascript
npm install
```

# Run

Npm scripts to run the interactive excercises are listed on the package.json

```javascript
//To visualize the exercise agenda
npm run <excercise-name>

//To submit your solution
npm run <execercise-name> verify <file-path>
```
